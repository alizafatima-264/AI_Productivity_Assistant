# AI Productivity Assistant

An AI-powered productivity application that helps users plan, organize, and achieve their goals through personalized roadmaps, focus sessions, and progress tracking.

## Overview

The AI Productivity Assistant is designed to help users overcome procrastination and stay consistent with their goals. Instead of functioning as a simple to-do list, it generates structured action plans, encourages focused work sessions, and tracks progress throughout the journey.

## Features

- AI-generated goal roadmap
- Smart goal creation
- Weekly planner
- Goal timeline
- Focus session timer
- Progress dashboard
- Goal categories
- Personalized scheduling
- Recovery mode
- Progress tracking

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Git & GitHub
- Visual Studio Code
- AntiGravity IDE
- ChatGPT

## How It Works

1. Create a goal.
2. Select the goal details and preferences.
3. Generate an AI-powered roadmap.
4. Follow the daily plan using focus sessions.
5. Track your progress on the dashboard.
